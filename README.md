# 📚 Books REST API

This project is a simple REST API built using Node.js and Express.

## 🚀 Features
- Get all books
- Get single book
- Add new book
- Update book
- Delete book

## 🛠 Tech Stack
- Node.js
- Express.js

## ▶️ Run Project
npm install
node server.js

## 📌 API Endpoints
GET /books
POST /books
PUT /books/:id
DELETE /books/:id

## 🧪 Testing
Use Postman to test all endpoints.
